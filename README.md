# Sentiment Analysis of IMDB Movie Reviews Using NLP

## Project Overview
This project performs sentiment analysis on IMDB movie reviews using Natural Language Processing techniques and Machine Learning algorithms.

## Dataset
IMDB Movie Reviews Dataset containing 50,000 movie reviews.

## Technologies Used
- Python
- Pandas
- NLTK
- Scikit-learn
- TF-IDF Vectorizer
- Logistic Regression
- Matplotlib
- Jupyter Notebook

## Workflow
- Data Loading
- Text Preprocessing
- Stopword Removal
- TF-IDF Vectorization
- Model Training
- Model Evaluation

## Results
- Model Accuracy: **88.74%**
- Correct Negative Predictions: **4328**
- Correct Positive Predictions: **4546**

## Conclusion
The Logistic Regression model achieved high accuracy and successfully classified movie reviews into positive and negative sentiments.
